# Country search application.
## Tech-stack:
 - Typescript
 - React
 - Redux Toolkit
 - MUI

<br />

### Features of the application
 * Change Theme
 * Display Country
 * Search for a specific country
 * Add and remove country from favourite list
 * Country details displayed in a different page
